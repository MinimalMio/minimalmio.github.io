记 在 Qemu 上安装 Windows NT Workstation 3.51 中文版

您需要准备

  • 一个最大 2GB 的空虚拟磁盘
  • 一个 Windows NT Workstation 3.51 安装光盘（ https://archive.org/details/WinNTWorkstation3.51CHS ）
  • 一张 DOS 启动软盘（要支持光盘的）

第一阶段

  • 使用 DOS 启动软盘引导系统
  • 使用 fdisk 分区
  • 使用 format c: 格式化
  • 进入 d:
  • 输入 winnt/b（可能需要先进i386文件夹）
  • 照常进行

第二阶段

  • 和 Windows NT 4.0 的第一阶段安装没有什么区别，略过
