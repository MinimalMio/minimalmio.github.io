解决 无法启动 X 服务器 问题

我在安装完毕 XRDP 后,尝试登录时,XRDP一直提示我“无法启动 X 服务器”
使用journalctl -xe查看日志,发现以下报错：xrdp-sesman[1156]: /usr/lib/xorg/Xorg.wrap: Only console users are allowed to run the X server
初步判定是/etc/X11/Xwrapper.config阻止了 XRDP 启动 X 服务器
发现最后一行写着allowed_users=console,将其修改为allowed_users=anybody,重启 XRDP 服务,恢复正常运行。
