QEMU Windows

如何为 Windows 98 启用 Qemu USB Tablet 支持

读前须知

  • 此教程已在 Windows 98 和 Windows ME 上测试过，但 Windows 95 未经测试

开始操作

第一步：先把你的 Windows 98 光盘上的内容复制到硬盘，一会会用

第二步：右键“我的电脑”，点击属性->设备管理器，找到 即插即用 BIOS 项（图标右下角会有一个警告图标）

第三步：把它的驱动程序更改为 PCI bus

第四步：系统会要求你重启，然后你会发现系统正在重装整个系统的驱动程序，因为此时你访问不了 CD-ROM，现在之前复制的安装文件就派上用场了

第五步：现在你的 Qemu USB Tablet 应该能正常工作了
